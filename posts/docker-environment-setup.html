Docker 环境安装 (Debian)

发表于2026-04-01|更新于2026-04-01|信息技术

Docker 是一种开源的容器化平台，用于将应用及其依赖打包到一个轻量级、可移植、自包含的运行环境中，从而实现“一次构建，到处运行”。它基于 Linux 内核的命名空间（namespace）和控制组（cgroups） 技术，实现进程级隔离和资源限制，比传统虚拟机更高效。

Docker 分为 Docker EE（企业版） 和 Docker CE（社区版），本文使用 Docker CE。

Docker Debian 安装文档：https://docs.docker.com/engine/install/debian

1. 安装基础工具

我使用 Debian 10 (Buster) 作为基础环境。如果你的服务器是初次使用那么需要先安装一些必须的依赖。

  1. 更新本地软件包索引列表。
  2. 安装必要依赖。
sudi apt update
sudo apt install ca-certificates curl

2. 配置 Docker GPG 密钥

  1. 创建一个目录用于存放 Docker GPG密钥，并给予 0755 权限。
  2. 从 Docker 官方地址下载 GPG密钥 保存到etc/apt/keyrings/docker.asc。
sudo install -m 0755 -d /etc/apt/keyrings
sudo curl -fsSL https://download.docker.com/linux/debian/gpg -o /etc/apt/keyrings/docker.asc

3. 配置密钥文件权限

给密钥文件加读权限，确保 APT 能够读取。

sudo chmod a+r /etc/apt/keyrings/docker.asc

4. 写入 APT 配置文件

把 Docker 官方软件源写入 APT 配置文件。

sudo tee /etc/apt/sources.list.d/docker.sources <<EOF
Types: deb
URIs: https://download.docker.com/linux/debian
Suites: $(. /etc/os-release && echo "$VERSION_CODENAME")
Components: stable
Architectures: $(dpkg --print-architecture)
Signed-By: /etc/apt/keyrings/docker.asc
EOF

5. 更新索引

再次更新索引，让 APT 识别新添加的 Docker 源。

sudo apt update

6. 安装 Docker 软件包

 sudo apt install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

7. 验证安装

如输出 Hello from Docker! ... 即表示 Dcoker 已经安装成功并正常工作。

sudo docker run hello-world

已成功安装 Docker Engine。
